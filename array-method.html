Array Methods:

  • forEach() method:(iterate through each element of an array)

     
                Example:
                    let array = [1,2,3,4,5];
                    array.forEach((element) => {
                        console.log(element);
                    });

            Example:
                    //same work with for loop

                    let array = [1,2,3,4,5];
                    for (let i = 0; i < array.length ; i++) {       // array.length is used to get the length of array, string, NodeList, etc (length is a property)
                        console.log(array[i]);
                    };

  • map() method:(create a new array with the results of calling a function for each element)

     
                Example:
                    let array = [1,2,3,4,5];
                    let result = array.map((element) => {
                        return element * 2;
                    });
                    
                    console.log(result);

  • push() method:(add value at the end of array)

     
                Example:
                    let array = [1,2,3,4,5];
                    array.push(6);
                    console.log(array); 
                    result = [1,2,3,4,5,6]  (is main 6 end per add ho gaya hy)

  • unshift() method:(add value at the start of array)

     
                Example:
                    let array = [1,2,3,4,5];
                    array.unshift(6);
                    console.log(array); 
                    result = [6,1,2,3,4,5];  (is main 6 start per add ho gaya hy)

  • pop() method:(last item/value will be remove)

     
                Example:
                    let array = [1,2,3,4,5];
                    array.pop();
                    console.log(array); 
                    result =[1,2,3,4];      (is main 5 remove ho gaya)

  • shift() method:(First item/value will be remove)

     
                Example:
                    let array = [1,2,3,4,5];
                    array.shift();
                    console.log(array); 
                    result =[2,3,4,5];      (is main 1 remove ho gaya)

  • splice() method:(array.splice(start, deleteCount, item1, item2, ...))

     
                Example:
                    let array = [1,2,3,4,5];
                    array.splice(1,1,6,7);      
                    console.log(array); 
                    result = [1,6,7,3,4,5];

  • slice() method:(array.slice(start, end))

    Note: slice () method is used with (string / array)

     
                Example:
                    let array = [1,2,3,4,5];        // if we need some part of array then we will use this method

                    // array.slice (0) index se start hoga aur array.slice(4) se end hoga but end wala index include nahi hoga

                    let newArray = array.slice (1, 3);  // slice means to copy or cut some piece of array or string
                    console.log( newArray); 
                    result = [2,3]

  • filter method

    Note: filter method condition check krta hy or true per kam krta hy 

                Example:
                    let array = [1,2,3,4,5];        

                    let newArray = array.filter((element) => {
                        return element > 3;
                        return element % 2 === 0;   // even num find out krna ho to is tarah se condition lagani pary gi                  

                    });

  • reduce method

    Note: reduce method array ke elements ko single value me convert krta hy 

                Example:
                    let array = [1,2,3,4,5];        

                    let result = array.reduce((accumulator, currentValue) => {
                        return accumulator + currentValue;
                    }, 0);   // 0 is the initial value of accumulator
                    console.log(result); 
                    result = 15;   (is main array ke sare elements add ho gaye)

  • concat() method:(to merge two or more arrays)

     
                    Example:
                        let array1 = [1,2,3];
                        let array2 = [4,5,6];
                        let result = array1.concat(array2);
                        console.log(result); 
                        result = [1,2,3,4,5,6];  (is main array1 aur array2 merge ho gaye)

  • indexOf() method:(to find the index of an element in an array)

     
                    Example:
                        let array = [1,2,3,4,5];
                        let index = array.indexOf(3);
                        console.log(index); 
                        result = 2;  (is main 3 ka index 2 hai kyuki array me counting 0 se start hoti hai)

  • includes() method:(to check if an element is present in an array)

     
                    Example:
                        let array = [1,2,3,4,5];
                        let result = array.includes(3);
                        console.log(result); 
                        result = true;  (is main 3 array me present hai to result true aayega)

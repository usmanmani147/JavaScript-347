Introduction to String Methods:

In JavaScript, strings are a fundamental data type used to represent text. String methods are built-in functions that allow you to manipulate and work with strings in various ways. These methods can be used to perform operations such as searching, slicing, replacing, and formatting strings.

String Methods:

String methods are built-in functions that allow you to manipulate and work with strings in JavaScript. Here are some commonly used string methods:

            
    
                    
    String Method:
    
                
  • str.slice(start, end)
    • 

                Example:

                let str = "usman";
                let slicedStr = str.slice(2, 5);
                console.log(slicedStr); // Output: "sma"
                

                
  • str.toUpperCase ()
    • 

                Example:

                let str = "usman";
                let newStr = str.toUpperCase();
                console.log(newStr); // Output: "USMAN"

                
  • str.toLowerCase()
    • 

                Example:

                let str = "USMAN";
                let newStr = str.toLowerCase();     
                console.log(newStr); // Output: "usman"

                
  • str.trim()
    • 

                Example:

                let str = "   Hello World!   ";
                let trimmedStr = str.trim();
                console.log(trimmedStr); // Output: "Hello World!"

                
  • str.slice(start, end)
    • 

                Example:

                let str = "usman";
                let slicedStr = str.slice(2, 5);
                console.log(slicedStr); // Output: "sma"

                
  • str1.concat(str2)
    • 

                Example:

                let str1 = "Hello,";
                let str2 = " World!";
                let concatenatedStr = str1.concat(str2);
                console.log(concatenatedStr); // Output: "Hello, World!"

                
  • str.replace(searchValue, newValue)
    • 

                Example:

                let str = "Hello, World!";
                let replacedStr = str.replace("World", "JavaScript");
                console.log(replacedStr); // Output: "Hello, JavaScript!"

                
  • str.replaceAll(searchValue, newValue)
    • 

                Example:

                let str = "Hello, World! Hello, World!";
                let replacedAllStr = str.replaceAll("World", "JavaScript");
                console.log(replacedAllStr); // Output: "Hello, JavaScript! Hello, JavaScript!"

                
  • str.split(separator)
    • 

                Example:

                let str = "apple,banana,cherry";
                let splitStr = str.split(",");
                console.log(splitStr); // Output: ["apple", "banana", "cherry"]

                
  • str.split(separator)
    • 

                Example:

                let str = "usman";
                let newStr = str.split("");
                console.log(newStr); // Output: ["u", "s", "m", "a", "n"]


                
  • \n (New Line Character):
    • 

                Example:

                let str = "Hello\nWorld!";
                let result = str;
                console.log(result); // Output: "Hello
                World! (is main Hello aur World alag line per print ho ga) 

                

                
  • \t (Tab Character):
    • 

                The \t character is used to create a horizontal tab in a string.

                Example:

                let str = "Hello\tWorld!";
                let result = str;
                console.log(result); // Output: "Hello    World! (is main Hello aur World ke darmiyan tab space hoga)   

                
  • str.charAt(index)
    • 

                The charAt() method returns the character at the specified index in a string.

                Example:

                let str = "usman";
                let char = str.charAt(2);
                console.log(char); // Output: "s"

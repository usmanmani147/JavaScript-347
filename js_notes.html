    • Attributes:

      Attributes are used to provide additional information about an element. 

     
            They are always specified in the opening tag of an element and usually come in name/value pairs like name="value".

            Example:

                <a href="https://www.google.com" target="_blank"> Google</a>

                In this example, "href" is an attribute that specifies the URL of the page the link goes to,
                and "target" is an attribute that specifies where to open the linked document.

            setAttribute() method is used to set the value of an attribute on the specified element.

            Example:

                let link = document.querySelector("a");

                link.setAttribute("href", "https://www.facebook.com"); 

                //Here, we are changing the href attribute of the link to "https://www.facebook.com".

                link.setAttribute("target", "_blank");    

                //Here, we are changing the target attribute of the link to "_blank" which means it will open in a new tab.

            getAttribute() method is used to get the value of an attribute on the specified element.

            Example:

                let link = document.querySelector("a");

                let hrefValue = link.getAttribute("href"); 

                console.log(hrefValue);     

                //Here, we are getting the value of the href attribute of the link and printing it in the console.

    • Events:

      Events are actions that occur in the browser, such as a user clicking a button, hovering over an element, or submitting a form.

      Events in javaScript (The change in the state of an object is known as an Event)

                Note: We can use JavaScript to respond to these events and create interactive web pages.

            Example:

                let button = document.querySelector("button");

                button.addEventListener("click", function() {
                    alert("Button was clicked!");
                });

                //Here, we are adding a click event listener to the button. When the button is clicked, 
                an alert will pop up with the message "Button was clicked!".

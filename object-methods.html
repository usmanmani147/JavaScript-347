•  Object Methods:

     Method 1:    Object.keys()    // yeh method object ke sari keys print karta ha.

    
            Example:
                    const student = {

                    name : "usmanghani",
                    age : 33,
                    class : "learning",
                    isPass : true,     
                    };

  •  Method 2:    Object.values()    // yeh method object ke sari values print karta ha.

    •             Example:    
                    const student = {
                    name : "usmanghani",
                    age : 33,
                    class : "learning",
                    isPass : true,     
                    };

                console.log(Object.values(student));  // Object values Method // note "O" of object will be capital

                // Output: ["usmanghani", 33, "learning", true];

  •  Method 3:    Object.entries()    // yeh method object ke sari entries print karta ha.

    •             Example:        
                    const student = {
                    name : "usmanghani",
                    age : 33,
                    class : "learning",
                    isPass : true,     
                    };

                console.log(Object.entries(student));  // Object entries Method // note "O" of object will be capital

                // Output: [["name", "usmanghani"], ["age", 33], ["class", "learning"], ["isPass", true]];

  •  Method to change values of objects:

    • 
            Example:    
                    const obj = {
                    name : "usman",
                    age : 33,
                    class : "learning",
                    isPass : true,     
                    };  


            obj.name = "ghani";     //1st method to modify the value of obj;

            console.log(obj.name);

            obj["age"] = 34;       //2nd method to modify the value of obj;

            console.log(obj["age"]);

            obj.name = obj.name + " ghani";     //3rd method to add value (without replacing existing) in object key (concatenation method)

            obj.name += " ghani";               //same as up method but this is the short cut of up method. (assignment operator)
